# Diabetes Prediction System

This project predicts whether a person has diabetes using Machine Learning.

## Features
- Data preprocessing
- Random Forest model
- Gradio web interface
- Hugging Face deployment

## How to Run
1. Install requirements:
pip install -r requirements.txt

2. Run app:
python app.py
